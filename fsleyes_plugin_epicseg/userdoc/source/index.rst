Overview
--------

This is the user documentation for *EpicSeg Project*.

This repository provides a FSLeyes plugin to automatically segment epicardial, paracardial fat on four chamber cine MRI using a optimised U-Net.

The plugin has been tested on FSLeyes version 0.34.2.


*FSLeyes* is the `FSL <http://fsl.fmrib.ox.ac.uk/fsl/fslwiki/>`_ image viewer.

You can download the latest version of FSLeyes from `FSLeyes <https://fsl.fmrib.ox.ac.uk/fsl/fslwiki/FSLeyes>`_
home page.

For more information on installing this software, please refer to `Installing FSLeyes doc <https://users.fmrib.ox.ac.uk/~paulmc/fsleyes/userdoc/latest/install.html>`_.

If you are getting started with FSLeyes, check out the `Quick start <https://users.fmrib.ox.ac.uk/~paulmc/fsleyes/userdoc/latest/quick_start.html>`_ page.
